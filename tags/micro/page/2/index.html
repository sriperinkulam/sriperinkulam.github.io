2024 12 01 003000

Moved over the syndication links in my posts to post-meta. The syndication icon only shows up if a valid link is present in the FrontMatter of a post. Once micro.blog picks up the feed, its currently set to auto publish to the fediverse on my GoToSocial (GTS) instance. On the backend, a GO script fetches the RSS feed from my GTS instance and saves the RSS feed item link and the corresponding link to my post in a JSON file. Another script quickly looks up which posts FrontMatter needs to be updated for the syndication links in each post and updates the files as needed. A bit windy but seems to do the job for now. I might end up consolidating these into just one script later and/or directly use the masto API to publish to the fediverse and grab the syndication link from there. For now, Im pretty psyched this is all working. ...
